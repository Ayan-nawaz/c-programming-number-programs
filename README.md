# C Programming - Number Programs

A collection of beginner-friendly C programs based on number operations. This repository helps beginners understand loops, functions, and logical thinking through common number-based programming problems.

## 📚 Programs Included

- Count Digits
- Sum of Digits
- Reverse Number
- Factorial
- Fibonacci Series

## 🎯 Topics Covered

- Number Manipulation
- Loops (`for`, `while`)
- Functions
- Recursion (where applicable)
- Mathematical Logic
- Problem Solving

## 🚀 Getting Started

1. Clone this repository:
   ```bash
   git clone https://github.com/Ayan-nawaz/c-programming-number-programs.git
   ```

2. Open the project in Visual Studio Code.

3. Compile any program:
   ```bash
   gcc filename.c -o output
   ```

4. Run the program:
   ```bash
   ./output
   ```

## 🛠️ Requirements

- GCC Compiler
- Visual Studio Code (Recommended)
